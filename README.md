## Vision

The vision of Projekt Wolkenschloss (German for "project cloud castle") is to be a complete self-hosting solution that provides the basic services you need in a personal cloud.
Wolkenschloss enables self-hosting by providing a curated, opinionated platform that aims to "just work". While technical enthusiasts can extend and customize everything, everyday users can just use it, knowing that privacy-respecting alternatives to Big Tech services are running securely in their own digital cloud castle.

## Project Status

The project is currently in an infant stage. Though our vision is grand, we are starting small with the most important feature: backups.
We have not yet released a stable version and you should probably not use Wolkenschloss as your only source of hosting and backup of your data until we have a stable release.
  
Also, the project is currently only a hobby of the maintainers. With we, I mean myself, Max (Geothain), currently. I am looking for maintainers to help me build this project. If you are interested, please check out our [contributing guide](CONTRIBUTING.md) and reach out to me.
