# Licensing

**Status: Superseded by [ADR 004 - License Change to AGPL-3.0](004-license-change-to-agpl.md)**

Which license should I use and why?

After reading <https://fossa.com/blog/all-about-copyleft-licenses/>,
<https://opensource.guide/legal/#which-open-source-license-is-appropriate-for-my-project>
and scrolling over <https://choosealicense.com/licenses/>, I decided to use
the MPL 2.0.

The main reasons are

- I want improvements to be shared with the open source community
- I believe in a collaborative approach to building software
- I might want to commercialize the project to be able to sustain and live from it (but not get rich)
- I don’t want the code made proprietary by someone else
