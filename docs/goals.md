# Our Goals

## Privacy by Design

- **Data Ownership**: Your data stays on your hardware, under your control
- **Zero Tracking**: No telemetry, analytics, or data collection by Wolkenschloss
- **Optimized for privacy**: We pick and optimize services and applications to be as private as possible

## Security by Design

- We pick and optimize services and applications to be as secure as possible
- **Backups included**: Comes with pre-configured backup systems that are maintained as well

## Curated Software

- **Curated software**: The maintainers select applications and services for common problems (e.g. like a photo cloud). Enthusiasts can extend, change and customize.
- **Unified Management**: Single interface for all services and system management

## Maintainability

Maintainability is a core goal of Wolkenschloss, because self hosting is only sustainable and attractive, if it is easy to maintain and update. We achieve this through:

- **Automatic Updates**: Security patches, fixes and improvements are automatically installed
- **Reproducible**: Using NixOS and GitOps and following the "Erase Your Darlings" principle
- **Modularity**: Easier to understand, modify and test and enables customizations
- **Transparent Operations**: Fully fledged monitoring stack
- **Comprehensible Architecture**: Prioritizing clarity over cleverness

## Accessible

- **High quality documentation**: Explaining concepts, architecture, motivation, goals and the technology
- **Consumer-Friendly**: Easy deployment and updates for non-technical users
- **Customizable**: Enabling customization for enthusiasts
- **Developer-Friendly**: Open architecture for community contributions
