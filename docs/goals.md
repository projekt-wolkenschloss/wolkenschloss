# Our Goals

In no specific order, our goals are:

## Privacy by Design

- **Data Ownership**: Your data stays on your hardware, under your control
- **Zero Tracking**: No telemetry, analytics, or data collection by the Wolkenschloss maintainers
- **Optimized for privacy**: We pick and optimize services and applications to be as private as possible

## Security by Design

- **Secure Applications**: We pick and optimize services and applications to be as secure as possible
- **Backups included**: Comes with pre-configured backup systems
- **Minimal Attack Surface**: Only necessary services and applications are included and exposed, and they are configured securely

## Maintainability

Maintainability is a core goal of Wolkenschloss, because self hosting is only sustainable and attractive, if it is easy to maintain and update. We want to achieve this through:

- **Automatic Updates**: Security patches, fixes and improvements are automatically installed
- **Reproducibility**: Using NixOS and GitOps principles
- **Modularity**: Easier to understand, modify and test and enables customizations
- **Transparent Operations**: Fully fledged monitoring stack
- **Comprehensible Architecture**: Prioritizing clarity over cleverness
- **Maintainable project and code**: Prioritizing readability and simplicity

## Accessibility

- **Good documentation**: Explaining concepts, architecture, motivation, goals and the technology
- **Consumer-Friendly**: Easy installation and updates for non-technical users
- **Easy recovery**: In case of hardware failures, the system can be easily recovered
- **Mobile Access**: Native mobile apps for key services
- **Multi User Support**: Multi-user support with appropriate permissions

## Curated Software

- **Curated Applications**: The maintainers select applications and services for common problems (e.g. like a photo cloud). Enthusiasts can extend, change and customize.
- **Unified Management**: Single interface for all services and system management
